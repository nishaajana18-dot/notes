Exponentiation by Squaring — Summary

Goal: Compute x^n efficiently.

Key ideas:
- Naive recursion: O(n) multiplications.
- Use even-power identity: x^n = (x^(n/2))^2 to halve the exponent each step.
- For odd n, compute x * x^(n-1) or use x * exp_by_squaring(x*x, (n-1)/2).
- Handle negative exponents by computing exp_by_squaring(1/x, -n).

Algorithm complexity:
- Time: O(log n)
- Space: O(log n) recursion depth (iterative variants reduce space)

Pseudocode:
function exp_by_squaring(x, n):
  if n < 0: return exp_by_squaring(1/x, -n)
  if n == 0: return 1
  if n is even: return exp_by_squaring(x*x, n/2)
  else: return x * exp_by_squaring(x*x, (n-1)/2)

Notes:
- Number of multiplications ≈ number of bits in n + number of 1s in n's binary form.
- Prefer iterative implementation in languages/environments sensitive to recursion depth.
